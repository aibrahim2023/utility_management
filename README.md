# utility_management
This project / application will be used to simulate a user's utility account to ensure payments are on time, and tracks any increases / decreases as well as energy utilization.


This line is a test to see if changes are updated in the repository.

This project will manage utilities such as:
- Eversource
- Nationalgrid

For each utility, it will provide the user information such as:
- utility type
- account number
- account owner (may be unnecessary)
- monthly price
- due date
- account status
